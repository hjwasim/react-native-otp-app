## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/react-native-otp-app.git
cd react-native-otp-app
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Firebase

- Download your `google-services.json` from the [Firebase Console](https://console.firebase.google.com/).
- Place it in the root of the project.

### 4. Prebuild and Start the App

```bash
npx expo pre-build --clean
npx expo start --dev-client
```
